# Claude Omni-Toolkit 🛠️

A centralized, community-driven repository of essential resources, prompts, scripts, and workflows for maximizing your productivity with the Anthropic Claude ecosystem.

Whether you are using **Claude Chat** on the web, automating your terminal with **Claude Code**, or orchestrating complex applications with **Claude Cowork**, this repository provides the "missing manual" and templates you need to get the most out of Claude.

## 🚀 Repository Structure

The toolkit is divided into three main pillars based on the Claude interface you are using:

### 1. 💬 [Claude Chat](./claude-chat)
A curated library of advanced prompts and system personas. Stop writing prompts from scratch.
* `/prompts`: Specialized templates for coding, writing, analysis, and data science.
* `/system-instructions`: Base personas to inject into your projects to permanently change how Claude behaves.

### 2. 💻 [Claude Code](./claude-code)
Scripts, tools, and configurations to extend Claude's capabilities in your local terminal.
* `/mcp-servers`: Guides and scripts to easily connect Claude Code to databases, Jira, GitHub, and local APIs via the Model Context Protocol (MCP).
* `/scripts`: Useful Bash/PowerShell aliases and automation scripts utilizing `claude-code`.
* `/config`: Example `CLAUDE.md` rule files designed for specific tech stacks (e.g., Next.js, Python, Rust).

### 3. 🤝 [Claude Cowork](./claude-cowork)
Workflows and instructions for building powerful AI assistants that navigate your desktop and browser.
* `/workflows`: Multi-step automation templates describing how to chain tools together effectively.

## 🛠️ Getting Started
Simply browse the directories above and copy what you need! Most folders contain their own `README.md` with specific installation or usage instructions for the assets inside.
